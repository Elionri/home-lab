Personal home laboratory for learning Security Operations Center (SOC), threat detection and incident analysis.

## About

**Цель проекта:** Демонстрация навыков построения инфраструктуры информационной безопасности и базового анализа событий на позицию SOC Analyst L1.

## Описание

Данный проект представляет собой локальную лабораторную среду (Home Lab) на базе ОС Ubuntu, развернутую для изучения работы SIEM-системы Wazuh. В рамках проекта настроен агентский сбор логов, централизованное управление политиками безопасности, а также базовые сценарии корреляции для обнаружения грубых атак (брутфорс, сканирование портов, изменения критичных файлов).

*Проект является демонстрационным и развернут в изолированной среде.*



## 🛠 Стек технологий / Структура
*Проект построен по модели "Агент-Сервер".*

| Компонент | Технология | Назначение |
| :--- | :--- | :--- |
| **ОС Хоста** | Ubuntu 22.04 LTS (Server) | Базовая платформа для развертывания серверной части |
| **SIEM / XDR** | Wazuh 4.7 | Центральный менеджер сбора событий, движок анализа и веб-интерфейс (Wazuh Dashboard) |
| **Индексация** | Elasticsearch OpenSearch | Хранение и индексация полученных логов для быстрого поиска |
| **Визуализация** | Wazuh Dashboard (Kibana) | Интерфейс для мониторинга и построения дашбордов |
| **Тестовые агенты** | Ubuntu (клиент) | Машины-жертвы для генерации телеметрии (брутфорс SSH, запуск вредоносных скриптов) |

##  Мануал по развертыванию (Installation & Run)

*Ниже описан способ установки вручную (All-in-One). Это один из самых простых способов поднять лабораторию для тестирования.*

###  Требования к окружению
- **ОС:** Ubuntu 22.04 LTS / 24.04 LTS (чистая установка)
- **Ресурсы:** Минимум 4 ГБ ОЗУ, 2 vCPU, 10 ГБ диска.
- **Доступ:** Прямой доступ к интернету (для скачивания пакетов) или настроенный прокси.
