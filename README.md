Personal home laboratory for learning Security Operations Center (SOC), threat detection and incident analysis.

## About

Цель проекта: Демонстрация навыков построения инфраструктуры информационной безопасности и базового анализа событий на позицию SOC Analyst L1.

## Описание

Данный проект представляет собой локальную лабораторную среду (Home Lab) на базе ОС Ubuntu, развернутую для изучения работы SIEM-системы Wazuh. В рамках проекта настроен агентский сбор логов, централизованное управление политиками безопасности, а также базовые сценарии корреляции для обнаружения грубых атак (брутфорс, сканирование портов, изменения критичных файлов).

Проект является демонстрационным и развернут в изолированной среде.
